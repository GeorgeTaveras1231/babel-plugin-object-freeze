## Example transpilation

Input:

```js
const config = {
  firstThreeAlphabetLetters: [
    'a',
    'b',
    'c'
  ]
};

```

Output:

```js
const config = Object.freeze({
  firstThreeAlphabetLetters: Object.freeze([
    'a',
    'b',
    'c'
  ])
});

```
